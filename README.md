# Retail EDA Project

Exploratory Data Analysis (EDA) on Retail Sales and McDonalds Menu Dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Datasets
1. Retail Sales Dataset
2. McDonalds Menu Dataset

## Project Objectives
- Perform data cleaning and exploration
- Analyze sales patterns
- Study customer demographics
- Visualize trends using charts

## Analysis Performed

### Retail Sales Dataset
- Dataset preview and structure
- Descriptive statistics
- Missing value analysis
- Gender distribution
- Product category distribution
- Sales trend over time
- Revenue by product category

### McDonalds Menu Dataset
- Dataset preview and structure
- Nutritional statistics
- Calories by category
- Calories distribution analysis

## Key Insights
- Clothing category recorded the highest purchases
- Electronics and Beauty categories contribute significantly to sales
- Most customers belong to the 20–50 age group
- McDonalds menu categories show large variation in calorie content
